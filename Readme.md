# Services Landing Template üíº

A clean, modular, and responsive HTML template focused on service-based websites. Ideal for consultants, advisors, or small businesses. It includes:

- Dynamic hero section
- WhatsApp floating button
- FormSubmit-ready contact form
- Neutral styling and easy theming
- Social media icons via SVG (editable)

## üîß Features

- üß± Clear and modular HTML structure  
- üéØ Primary focus on services (consulting); secondary on product promotion  
- üìê Prebuilt sections: header, hero, services, about, contact, social, footer  
- üñºÔ∏è Hero supports dynamic background or image banner (`data-hero-type`)  
- üé® Color variables defined in `:root` (easy theming via `theme.css`)  
- üÜò Logo fallback: text is shown if the image fails to load  
- üí¨ Floating WhatsApp button with optional popup  
- üåê Ready-to-use social media SVG buttons (Instagram, TikTok, Facebook, etc.)  
- üì® Contact form compatible with [FormSubmit.co](https://formsubmit.co) ‚Äì no backend required  
- üì± Fully responsive and accessible CSS design  
- üöÄ Built for quick customization and fast deployment  

## üß∞ How to Use

1. **Clone or fork this repository**

```
git clone https://github.com/QuantumRevenant/services-landing-template.git
cd services-landing-template
```

2. **Edit the content** in `index.html`, `style.css`, and optionally `script.js`  
3. **Replace or customize hero images** in `/resources/hero_images/`  
4. **Edit root colors** in `:root` or override via `theme.css`  
5. **Connect FormSubmit** by setting your form's `action` URL  
6. **Deploy to GitHub Pages**, Netlify, Vercel, or any static host
